TROUBLESHOOTING
================================================================================

Problem: "Permission denied" when pushing to GitHub
Solution: Use Personal Access Token instead of password
         https://github.com/settings/tokens

Problem: "Package not found" on Packagist
Solution: Wait 5-10 minutes, Packagist needs time to index

Problem: Webhook not working
Solution: Check GitHub webhook delivery status
         Settings â†’ Webhooks â†’ Recent Deliveries

Problem: composer require fails
Solution: Make sure you tagged the release (v1.0.0)
         Check minimum-stability in composer.json
